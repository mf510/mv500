MV500: Master Verbs 500
MV500 is a standalone dataset and engine designed to master the "skeleton" of the English language: verb conjugations. It covers 500 essential core verbs, structured progressively from Lv100 to Lv500, targeting learners from beginner to intermediate levels (EIKEN Pre-2 to 2).
Learning Framework:
Each verb is tagged with a proficiency indicator (?, △, ◯) for self-assessment. Designed to provide a clear roadmap from foundational verbs to those used in broader logical discourse.
Concept:
A "pure English logic engine" that operates offline with zero external dependencies. Built for speed, ultra-low power consumption, and total privacy.
